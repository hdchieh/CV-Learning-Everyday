# CV-Learning-Everyday

本项目是自己对于CV领域一些论文的链接汇总和相关论文的学习笔记。

文章均采用Markdown格式书写，公式使用LATEX编辑。

PPT文件夹包含读博组会时分享的PPT,一般以论文阅读笔记为主。
